# ATM_CONSOLE_BASED_PROJECT

ATM System in Python

Description

This is a simple ATM system implemented in Python. The program allows users to perform basic banking operations such as withdrawing money, depositing funds, generating a PIN, and viewing a mini statement.

Features

Withdraw money (requires PIN authentication)

Deposit funds

Generate a PIN for security

View mini statement (account details and balance)

Exit the system

How to Use

Run the Python script.

Select an option from the menu.

Enter the required details like account number, PIN (if required), and amount.

The system will validate the details and perform the requested operation.

Requirements

Python 3.x
